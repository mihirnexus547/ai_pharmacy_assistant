How to link your Twilio number

Our backend serves an XML TwiML endpoint that connects Twilio incoming voice streams to a fast, bidirectional WebSocket connection.

1

Expose your application

Expose port 8000 using ngrok http 8000 or deploy the application to a cloud provider like Railway.

2

Configure the Webhook

Go to your Twilio Console > Active Numbers, and set "A Call Comes In" webhook to https://your-public-url.ngrok-free.app/twilio/inbound as HTTP POST.

3

Place a call

Dial the Twilio number! The agent will answer instantly, transcribing and replying over the phone line in real-time.
